# Frontend Polish Skill

Use this skill when improving UI quality, responsiveness, spacing, typography, component hierarchy, and interaction states.

## Checklist
- Check spacing consistency.
- Check responsive behavior.
- Check hover, focus and active states.
- Check empty states.
- Check loading states.
- Check error states.
- Check mobile layout first.
- Avoid excessive animations.
- Prefer purposeful transitions.
- Ensure buttons have clear visual hierarchy.
- Ensure forms are easy to scan.
- Ensure sections have different rhythms.

## Required output
Before implementing:
1. Identify weak UI areas
2. Suggest concrete improvements
3. Explain expected visual impact
4. Confirm which files will be changed

After implementing:
1. Summarize changes
2. Mention responsive behavior
3. Mention remaining visual risks
